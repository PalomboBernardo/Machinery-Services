# 🌾 Machinery Services

**Machinery Services** es una página web pensada para facilitar la reserva y contratación de maquinaria agrícola.

- 🌐 Página responsive 
- 📋 Formulario de reserva de maquinaria
- 📞 Información de contacto

## 🛠️ Tecnologías utilizadas

- HTML5
- SCSS 
- Git para control de versiones
